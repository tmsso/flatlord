Key decisions (short rationale)
  • Accent — deep teal oklch(0.47 0.085 205): confident without reading "default Office blue"; keeps semantic info-blue free; AA in both themes. MS blue & slate-indigo remain one-line token swaps (switcher in 01).
  • Type — IBM Plex Sans (+ Plex Mono): enterprise-clarity grotesque, full Hungarian diacritics (ő ű), true tabular figures for money/meter columns; mono for serials.
  • Density — two registers: admin 36px rows / 13px table text; tenant 44px+ targets, one primary action per screen. Same tokens.
  • Structure over flatness: warm-grey background, white surfaces, real 1px borders + subtle shadows everywhere; dark theme keeps the same borders on warm dark greys — no pure-black voids.
  • Status = badge + icon + label, never colour alone; no ALL-CAPS anywhere (diacritics).
